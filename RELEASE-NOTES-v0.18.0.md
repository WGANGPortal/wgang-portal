# WGANG Portal v0.18.0 – Chill Bunny Planner

## Nytt
- Visuell Chill Bunny-oppgavetavle med 12 aktive oppgavekort.
- Popularitetsfarge som viser samlet etterspørsel uten å stoppe ved 5 valgte: 0, 1–2, 3–4, 5–6, 7–8 og 9+ valgte.
- Fem kan ta samme oppgave samtidig, men kortene kan ha høyere interesse fordi rask gjennomføring gjør at flere kan rekke samme oppgave i samme 10-minutters harepust.
- Tydelig tips om at en oppgave ikke bør tas før spilleren er klar til å fullføre den umiddelbart.
- Medlemmer kan velge «Jeg har den klar», «Jeg klargjør den» eller «Ikke aktuell».
- Personlig oversikt over klargjorte oppgaver, sortert med mest populære først.
- Varsel dersom tavlen ikke er bekreftet oppdatert etter daglig oppgavebytte kl. 10:00.
- Oppgavebibliotek som kan utvides når nye oppgaver dukker opp.
- Eier, Admin og Ass. leder kan velge og publisere dagens oppgavetavle.
- 12 kjente Chill Bunny-oppgaver er lagt inn som startbibliotek.

## Viktig før opplasting
Kjør `MIGRATION-v0.18.0-CHILL-BUNNY.sql` i Supabase SQL Editor først. Deretter kan hele portalpakken lastes opp til GitHub.
