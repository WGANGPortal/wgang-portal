# WGANG Portal v0.8 – Shared Data Foundation

Dette er første versjon som er klargjort for ekte flerbrukerdrift.

Når Supabase er konfigurert, deles følgende mellom alle godkjente medlemmer:

- medlemskonto og Hay Day-navn
- roller og godkjenningsstatus
- deltakelse i neste derby
- oppgavepreferanser
- derbytype, poengkrav og strategi

Versjonen beholder lokal demomodus slik at portalen fortsatt kan forhåndsvises før databasen er koblet til.

Se `SETUP-SUPABASE.md` før produksjonsopplasting.
