# WGANG Portal v0.17.1.2 – Profile Menu & Logout Fix

- Profilmenyens undersider vises som separate visninger.
- Innstillinger viser bare innstillingsseksjonen.
- Se / endre profil viser bare profilseksjonen.
- Min konto viser bare kontoseksjonen.
- Varsler viser bare varslingsseksjonen.
- Logg ut bruker nå portalens faktiske `backend.signOut()`-flyt.
- Etter utlogging vises landingssiden som utlogget.
- Det unødvendige pilikonet i Logg ut-raden er fjernet.
