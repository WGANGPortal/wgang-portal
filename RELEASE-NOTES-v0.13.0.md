# WGANG Portal v0.13.0 – Leadership Chat & Language Polish

- New private Leadership Chat for Owner, Administrator and Assistant Leader.
- Supabase RLS protects leadership messages from ordinary members.
- Compact language selector using flags only (🇳🇴 / 🇬🇧).
- Expanded English UI translations and translation of known Derby templates, rules and strategy texts.
- Existing members, profiles, Derby data and community content are preserved.

Note: Free-form user-generated Norwegian text is still stored in its original language. Full automatic machine translation of arbitrary posts requires a translation provider/API and is not included in this release.
