# WGANG Portal v0.14.0 – Complete Language Support

Denne versjonen fokuserer på konsekvent norsk/engelsk visning.

## Endringer
- Utvidet engelsk oversettelse av systemstyrt innhold i hele portalen.
- Oppgavekategorier og oppgavenavn oversettes når engelsk er valgt.
- Derbytyper, statuser, standardregler og strategitekster får engelsk visning.
- Flere overskrifter, hjelpetekster, medlems- og admintekster er oversatt.
- Språkvelgeren beholdes som kompakt flaggvalg.
- Norsk er fortsatt standardspråk.
- Brukergenerert fritekst beholdes på originalspråket; automatisk maskinoversettelse er ikke koblet til ennå.

Ingen databaseendring er nødvendig for v0.14.0.
