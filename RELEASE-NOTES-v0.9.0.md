# WGANG Portal v0.9.0 – Community Content

Denne versjonen gjør WGANG Portal mer levende og lar nabolaget bidra med innhold.

## Nytt
- Medlemmer kan sende inn Tips & triks til godkjenning.
- Eier/Administrator kan godkjenne eller avslå innsendte tips.
- Eier/Administrator kan publisere egne tips direkte.
- Godkjente medlemmer kan publisere innlegg i Derbyprat.
- Eier/Administrator kan publisere kunngjøringer.
- Siste kunngjøring vises også på oversikten.
- Admin kan slette publisert innhold.

## Viktig før opplasting
Kjør `MIGRATION-v0.9.0-COMMUNITY-CONTENT.sql` i Supabase SQL Editor én gang før den nye nettsideversjonen lastes opp.
