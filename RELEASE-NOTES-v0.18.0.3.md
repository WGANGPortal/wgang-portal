# WGANG Portal v0.18.0.3

## Chill Bunny – publisert tavle og redigerbare standardkort

- Dagens 12 Chill Bunny-oppgaver brukes som startbibliotek og kan publiseres som dagens tavle.
- Standardkort viser kategori, ikon, antall, oppgavenavn og tidsfrist/varighet.
- Eier, Admin og Ass. leder kan redigere oppgavenavn, antall og tidsfrist på lagrede kort.
- Nye kort lagres i biblioteket og kan gjenbrukes på fremtidige tavler.
- Popularitetsnivåene fra v0.18.0.1 beholdes.
- Tavlen oppdateres manuelt etter spillets daglige oppgavebytte kl. 10:00.
