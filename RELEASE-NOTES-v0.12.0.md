# WGANG Portal v0.12.0 – Derby Roles & Management

## Nytt
- Ny rolle: Ass. leder (`assistant_leader`) mellom Medlem og Administrator.
- Ass. leder kan se den detaljerte oppgavepreferanseoversikten uten å få full admin-tilgang.
- Rolleadministrasjon støtter Medlem, Ass. leder, Administrator og Eier.
- Eierrollen er beskyttet mot utilsiktet endring fra Administrator.
- Derbyterminologi er oppdatert fra Bunny til Harepus via databasegrunnmalene.
- Harepusstrategi: klargjør oppgaver slik at de kan fullføres mens harepusen er aktiv og oppgavene er rosa.
- Eier kan lagre endringer permanent i en derbygrunnmal med «Lagre som standard».
- Admin kan fortsatt publisere justeringer som bare gjelder det aktuelle derbyet.
- Grunnmalen viser hvem som sist oppdaterte den og tidspunkt når informasjonen finnes.

Eksisterende medlemmer, profiler, innhold, derbydata og preferanser beholdes.
