# WGANG Portal v0.10.0 – Derby Management & Admin Alerts

## Nytt
- Derbyadministrasjon med grunnmaler og publisering av neste derby.
- Neste derby vises så snart admin publiserer det.
- Standard tidsplan: tirsdag kl. 10:00 til mandag kl. 10:00.
- Anbefalt svarfrist: mandag kl. 23:00.
- Chill Derby og Chill Bunny Derby støttes i grunnmalene.
- Deltakelse lagres per publisert derby.
- Adminvarsler for medlemsforespørsler og tips som venter på godkjenning.
- Adminoversikt viser hvilke medlemmer som liker og kan ta ulike oppgaver.
- Tomat lagt til under innhøstingsoppgaver.

## Bevarer eksisterende data
Oppdateringen sletter ikke medlemmer, roller, community-innhold eller eksisterende oppgavepreferanser.
