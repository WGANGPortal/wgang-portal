# WGANG Portal v0.11.0 – Profiles & Language Foundation

## New
- Voluntary member profiles visible to approved WGANG members.
- Members can share as much or as little as they want: About me, gender, age group, country/place, Hay Day experience and favorite part of the game.
- Hay Day names are normalized to uppercase for existing and future members.
- Profile cards use Hay Day names as the public identity.
- Email addresses are removed from portal UI and browser-readable profile data.
- Norwegian / English language selector added as an initial interface translation foundation.

## Privacy
- Email remains stored for authentication, but authenticated browser clients can no longer select the email column from `public.profiles`.
- Public member profiles never show email.

## Important note about translation
The English interface is an initial/beta translation layer for fixed portal text. User-generated content (announcements, Derby Talk and submitted tips) remains in its original language until a dedicated translation service is connected in a later update.
